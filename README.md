\# Portfolio Optimization Project



\## Time Series Forecasting for Portfolio Management Optimization



\### Overview

This project applies time series forecasting to historical financial data to enhance portfolio management strategies for GMF Investments.



\### Data Sources

\- \*\*TSLA\*\*: Tesla Inc. (High-growth stock)

\- \*\*BND\*\*: Vanguard Total Bond Market ETF (Low risk bonds)

\- \*\*SPY\*\*: S\&P 500 ETF (Moderate risk market index)



\### Project Structure

├── data/

│ └── processed/ # Cleaned data files

├── src/ # Reusable Python code

├── tests/ # Unit tests

└── scripts/ # Python scripts
